# Como rodar o projeto
#Clone o projeto
    $ git clone https://github.com/mtNeto07/Controle-Financeiro

    # Instale todas as depedências
    $ yarn

    # Rode o projeto
    $ yarn start

    # O servidor vai ser inicializado no <http://localhost:3000/>
